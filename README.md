# Island Adventures — Android APK

## Как собрать APK (пошаговая инструкция)

### Шаг 1 — Создай аккаунт GitHub
1. Зайди на **https://github.com**
2. Нажми **Sign up** (бесплатно)
3. Подтверди email

---

### Шаг 2 — Создай репозиторий
1. Нажми **+** → **New repository**
2. Название: `island-adventures`
3. Выбери **Private** (чтобы никто не видел)
4. Нажми **Create repository**

---

### Шаг 3 — Загрузи файлы
1. На странице репозитория нажми **uploading an existing file**
2. Перетащи ВСЕ файлы и папки из архива в браузер:
   - `main.py`
   - `buildozer.spec`
   - папку `assets/` со всеми картинками и музыкой
   - папку `.github/` (папка со скрытым именем!)
3. Нажми **Commit changes**

> ⚠️ Папка `.github` скрытая — убедись что она тоже загружена!

---

### Шаг 4 — Запусти сборку
1. Перейди во вкладку **Actions**
2. Увидишь процесс **Build APK** — он запустится автоматически
3. Подожди **15–30 минут** (первый раз дольше из-за скачивания Android SDK)
4. Зелёная галочка = успех ✅

---

### Шаг 5 — Скачай APK
1. Нажми на завершённый **Build APK**
2. Внизу страницы раздел **Artifacts**
3. Скачай **island-adventures-apk.zip**
4. Распакуй — внутри файл `.apk`

---

### Шаг 6 — Установи на телефон
1. Скопируй `.apk` на телефон (через USB или мессенджер)
2. На телефоне открой файл
3. Android спросит разрешение на установку из неизвестных источников
4. Разреши → Установить

---

## Управление в игре
- **Джойстик** (крестовина) — левый нижний угол экрана
- **🎒 Рюкзак** — правый верхний угол
- **☰ Меню** — правый верхний угол

## Требования к телефону
- Android 8.0 и выше
- ~100 МБ свободного места
